# Domain entities

Folder này dành cho entity nghiệp vụ thuần như `Tenant`, `Conversation`, `Message`, `Appointment`, `KnowledgeDocument`, `Staff` và `HandoffEvent`.

Prompt 03 chưa tạo entity giả để tránh phát sinh abstraction chưa được dùng. Khi tách thật, cần đối chiếu Prisma schema và response contract hiện tại trước.
